## 1️⃣Sacar Template strings ---> teclado (Latinoamericano)  
Para sacar esas comillas se llaman Acento grave  y se sacan con:
- en teclado **`ALT GR`** + **`}]`**  -------- _(comprobado)_   
- en teclado **`ALT`** + **`96`** ----------- _(pendiente)_
